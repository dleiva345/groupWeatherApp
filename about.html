IT 4203 Term Project

In this term project, we apply all the knowledge and skills developed through the individual Milestone projects and develop a single page weather application similar to the individual projects. The knowledge and skills applied include jQuery, JSON, RESTful Web API, AJAX, and rich UI design.

Project Details

Create a website that will allow the users to see the current weather and search a city or location to get the current and weather forecast for that location.

We used Bootstrap for the front-end UI
